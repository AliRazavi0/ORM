# ORM

Object–relational mapping

## Installation

Use the package manager [composer](https://getcomposer.org/) to install orm.



```bash
composer install 
```

## Usage

```php
create database 

edit file ORM/src/Configs/database.php 

add setting env system 

key array orm_test
```
